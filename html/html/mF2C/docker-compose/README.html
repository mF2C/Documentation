Deploying the mF2C System with docker-compose¶

Prerequisites¶

User that have docker and docker-compose installed on Mac, Windows or Linux systems (https://docs.docker.com/compose/install/).

Installing¶

With docker-compose, users can orchestrate the installation of the mF2C system by doing:

docker-compose -p mf2c up

The full installation might take a few minutes, depending on the user’s local Docker images and network connection
